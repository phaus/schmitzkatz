RXTX on OS X: Serial port already in use

2013-05-02
Tweet
After starting my Java app I got a message that the serial port I was trying to connect to was already in use. The problem turned out to be a completely different thing.

When I first got this message I was looking for the issue in my code. After 15 minutes of making sure this was not the case I searched the web and quickly found an answer. If you ever run into this issue make sure that a directory called /var/lock exists and that it has proper permissions set.

sudo mkdir /var/lock
sudo chmod a+rw /var/lock

I found the solution on Stackoverflow: RXTX on Mac - Serial port already in use.
